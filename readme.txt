README Albatir

Pour lancer le jeu vous aurez besoin des ressources présentent dans les 
3 dossiers "Jeu", "Menu", "ressources".

Vous pourrez lancer le jeu a partir du menu "Menu.html",
Ou directement via "Jeu.html"

Commandes: 
Z,Q,S,D pour déplacer le vaisseau,
ESPACE pour tirer,
N pour mettre en pause,
Le reste des commandes sont indiquées dans l'interface du jeu.
